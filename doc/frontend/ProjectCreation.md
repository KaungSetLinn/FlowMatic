# 新規プロジェクト作成

- **タイトル**：  
  （テキスト入力、必須）  
  プロジェクトの名称を入力。

- **説明**：  
  （テキストエリア、必須）  
  プロジェクトの目的や概要を入力。

- **開始日**：  
  （カレンダー＋時間入力、必須）  
  プロジェクトの開始日時を設定。  
  ISO形式で送信（例：`2025-10-24T09:00:00Z`）

- **締切日**：  
  （カレンダー＋時間入力、必須）  
  プロジェクトの終了予定日時を設定。  
  ISO形式で送信（例：`2025-12-31T23:59:59Z`）

- **ステータス**：  
  （セレクトボックス）  
  - 計画中（`planning`）  
  - 進行中（`in_progress`）  
  - 完了（`completed`）

- **メンバー**：  
  （複数選択）  
  プロジェクトに参加するメンバーを選択。  
  APIにはUUIDの配列として送信。


**progress はリクエストボディに含めず、サーバー側で自動的に 0 を設定する。**
